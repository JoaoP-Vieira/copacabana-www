# What?

Copacabana Linux, is an independent Linux(R) distribution focused on simplicity, sanity, modularity and liberty (as in Enlightenment).
It uses a *BSD-inspired distribution model, based in tarball stages, not on the "everything is a package idea", since it's simpler to package, distribute and maintain.

# Why?

- "Competition is good, principally if it's offering more freedom."

We always thought that brazilian distributions weren't exactly neither original nor independent, that's why we're here.
